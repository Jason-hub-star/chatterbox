# snack-web 이차창작(팬아트/동인) 정책

**시행일**: 2026-07-01
**최종 수정일**: 2026-07-01

## 1. 정책 개요

snack-web(이하 "서비스")은 사용자가 생성한 창작물을 존중하면서도, 원작자의 지적재산권을 보호합니다.

본 정책은 **원작 IP(저작권, 상표, 캐릭터)를 기반으로 한 이차창작(팬아트, 동인, 2차 저작물)**의 허용 범위를 정의합니다. 다만 이는 기술적 가이드라인이며, **서비스 출시 전 법률 자문을 통해 검증**됩니다.

## 2. 이차창작 정의

### 2.1 포함 범위

다음과 같은 콘텐츠를 "이차창작"으로 분류합니다:

| 유형 | 설명 | 예시 |
|------|------|------|
| **캐릭터 기반 창작** | 원작 캐릭터를 모티브한 신규 모델/버추얼 아바타 | 원작 캐릭터를 본떠 만든 버튜버 모델 |
| **음악 리믹스/편곡** | 원작 음악을 변형한 음악 | 유명곡 인스트루멘탈 편곡 |
| **영상 패러디** | 원작 영상 장면을 활용한 패러디 | 유명 영상에 더빙/자막 추가 |
| **설정 재창작** | 원작 배경/설정을 기반한 새로운 이야기 | 원작 세계관 팬픽 영상화 |
| **이미지 변형** | 원작 이미지를 기반한 팬아트 | 원작 캐릭터 그림 재해석 |

### 2.2 제외 범위

다음은 이차창작이 아니며, **직접 복제**로 간주됩니다:

- 원작 그래픽/영상 **그대로** 업로드 (변형 없음)
- 원작 음성/음악 **전곡** 무단 사용
- 원작 대사/텍스트 **그대로** 재배포

## 3. 허용/불허 기준

### 3.1 이차창작 "허용" 판정 기준

다음을 **모두** 만족하면 이차창작으로 간주되어 허용됩니다:

| 기준 | 설명 | 검증 방법 |
|------|------|----------|
| **외형 변형 70% 이상** | 캐릭터/이미지의 외형이 원작으로부터 명백히 구별됨 | 원작과 나란히 비교했을 때 즉시 다른 콘텐츠로 인식 가능 여부 |
| **비상업적 개인 창작** | 판매·수익화·상업적 이용 의도 없음 | 콘텐츠 설명, 링크, 수익화 여부 |
| **팬 오마주 명시** | "원작: [원작명]", "팬 창작" 명시 | 메타데이터, 제목, 설명에 출처 표기 |
| **원작사 명시적 금지 위반 아님** | 원작사가 공식 가이드에서 금지한 행위가 아님 | 원작사 공식 가이드라인 확인 |
| **인신공격·혐오 없음** | 원작자/원작 팬덤을 모욕하지 않음 | 콘텐츠 본문 검토 |

### 3.2 이차창작 "불허" 판정 기준

다음 중 **하나라도** 해당하면 삭제 대상입니다:

| 금지 행위 | 설명 |
|---------|------|
| **원작 그래픽 자체 복제** | 원작 모델/이미지를 그대로 또는 경미한 편집으로만 재사용 (외형 변형 70% 미만) |
| **상표/로고 도용** | 원작의 공식 로고, 상표명을 자신의 것처럼 표기 또는 판매 |
| **원작사 명시적 금지** | 원작사가 "이차창작 금지" 공시한 캐릭터·설정 |
| **상업적 이용** | 판매, 크라우드펀딩, 수익화 목적의 배포 |
| **원작자 모욕** | "원작은 쓰레기", "원작자는 사기꾼" 등 원작자를 貶損 |
| **허위 표기** | 자신의 창작물인 척 거짓 주장 또는 원작 표기 의도적 삭제 |

### 3.3 회색 지대 (검토 대상)

다음은 사건별·지역별 판단이 필요합니다. **신고 접수 시 개별 검토**됩니다:

- **외형 변형 50~69%**: 변형 정도와 원작과의 차이도를 상세 비교
- **비상업적 수익화**: 광고 수익, 후원, 후원팀 (협회 판단 대상)
- **설정만 차용**: 세계관/배경만 사용하고 캐릭터는 신규 (사실 파악 필요)
- **국가별 정책**: 일본 동인 문화, 한국 저작권법, 미국 공정 이용 기준 상이

## 4. 신고 및 삭제 절차

### 4.1 이차창작 신고

원작사·저작권자가 이차창작의 삭제를 요청하는 방법:

**이메일로 신고해주세요:**

```
받는 사람: gmdqn2tp@gmail.com
제목: [이차창작 삭제 요청] (원작명 / 이차창작 콘텐츠 ID)
```

**신고 내용에 포함해야 할 정보:**

1. **신고인 정보**
   - 성명 또는 법인명
   - 신고인이 원작권자임을 나타내는 증명 (저작권 등록증, 공식 계약서, 기업 인증)

2. **원작 정보**
   - 원작명 및 원작사
   - 저작권 표시 및 공식 게시물 링크

3. **이차창작 콘텐츠 정보**
   - snack-web 내 콘텐츠 URL 또는 ID
   - 업로드자 username
   - 침해 이유 (위 3.2 불허 기준 중 어떤 항목에 해당하는지)

4. **법적 선언**
   - 신고 내용이 사실이며 저작권자임을 서약

### 4.2 처리 절차

[COPYRIGHT-DMCA.md § 3 신고 처리 절차](COPYRIGHT-DMCA.md#3-신고-처리-절차)를 적용합니다.

| **초기 검토** | 7 영업일 | 이차창작 여부 판단 (허용/불허/회색지대 분류) |

**총 처리 기한**: 최대 30~45 영업일

- **재검토**: 운영자가 7 영업일 내 재검토 및 최종 판정

## 5. 이차창작 가이드라인 (이용자용)

이차창작을 계획 중인 크리에이터를 위한 Best Practice:

1. **원작사 공식 가이드라인 확인**
   - "팬 창작 허용" 공시 여부 확인 (예: Nintendo, 픽시브)
   - 명시적 금지 캐릭터·설정이 있는지 확인

2. **충분한 변형**
   - 원작과 즉시 구별 가능한 수준의 변형 (외형 70% 이상)
   - 새로운 표정, 색상, 의상, 맥락 추가

3. **출처 명시**
   - 제목 또는 설명에 `원작: [원작명]` 명시
   - 원작사 공식 계정 링크 포함 권장
   - "팬 창작", "이차창작", "팬아트" 태그 사용

4. **상업화 자제**
   - 비상업적 개인 창작 범위 내에서만 공개 배포
   - 후원/판매 의도가 있으면 원작사에 사전 문의

5. **커뮤니티 존중**
   - 원작자·원작 팬덤을 존중하는 태도 유지
   - 비판적 의견도 건설적 톤으로 표현

## 6. 지역별 참고 사항

### 6.1 일본 (동인 문화)

- 동인/팬 창작은 일본 저작권법상 관행적으로 관대한 편입니다.
- 다만 원작사의 "공식 가이드"를 따를 것을 권장합니다. (예: Nintendo, 애니플러스)
- 서비스는 "일본 동인 문화" 자체를 배제하지 않으며, 각 사건별 원작사 정책을 존중합니다.

### 6.2 한국

- 한국 저작권법은 이차창작에 대해 "공정이용" 기준이 엄격한 편입니다.
- 웹소설·웹툰 캐릭터 이용 시 원작사 동의 필수로 보는 경향이 있습니다.
- 개인 블로그 팬아트는 관행상 용인되지만, 플랫폼 배포 시 원작사 정책 확인 필수입니다.

### 6.3 미국 (공정이용 기준)

- Fair Use: 변형도, 목적(비상업), 원작 시장에의 영향, 사용 범위 종합 판단
- 팬 창작·패러디는 상대적으로 관대합니다.
