## Operating Rule

- `npm run docs:check`: 문서 구조 기본 게이트. 항상 PASS 상태를 유지한다.
- `npm run docs:check:strict`: 구현 착수 게이트. `계약 공백`, active PENDING, BLOCKED gap이 있으면 실패한다.
- `npm run docs:health`: 현재 계약망 상태를 사람이 읽는 리포트로 출력한다.
