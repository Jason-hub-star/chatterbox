# 커뮤니티 가이드라인 (G-121)

## §1 커뮤니티 가이드라인 개요

- **이 가이드라인의 목적**: 가상 공연 공간에서 모든 참가자가 안전하고 창의적으로 연기할 수 있도록
- **적용 대상**: 모든 회원 (방장·참가자·관전자)
- **시행 방법**: 신고 시스템 + 운영자 검토 + 자동 모더레이션

---

## §2 허용 콘텐츠

- 창작 연기, 즉흥극, 역할극 (픽션 맥락 명확한 경우)
- 성인 주제를 다루는 연극 콘텐츠 (단 성적으로 노골적이지 않은 범위)
- 다양한 언어·문화권 콘텐츠 (단 혐오 발언 제외)

---

## §3 금지 행동 — 즉시 정지 대상

### 콘텐츠 관련

- 성적으로 노골적인 콘텐츠 (미성년자 관련 무관용)
- 실제 인물을 모욕하거나 명예를 훼손하는 콘텐츠
- 자해·자살을 미화하거나 조장하는 콘텐츠
- 실제 폭력·범죄를 촉구하는 콘텐츠

### 행동 관련

- 다른 사용자 괴롭힘·스토킹·개인정보 노출(doxxing)
- 스팸·광고·피싱 링크
- 타 사용자 계정 도용·사칭
- 서비스 또는 타 사용자의 기술 인프라 공격

### AI 생성 콘텐츠 관련

- 실존 인물의 동의 없는 딥페이크 생성
- 저작권 보호 콘텐츠를 명백히 복제하는 프롬프트

---

## §4 신고 프로세스

- **신고 방법**: 채팅 메시지 우클릭 → 신고, 참가자 이름 클릭 → 신고
- **신고 분류**: 스팸·괴롭힘·부적절콘텐츠·사칭·기타
- **처리 타임라인**:
  - 접수 즉시: 자동 확인 이메일 (gmdqn2tp@gmail.com → 검토)
  - 48시간 이내: 운영자 초기 검토
  - 7일 이내: 조치 완료 + 결과 통보

---

## §5 조치 단계

1. **경고**: 최초 위반 (경미), 이메일 통보
2. **일시 정지**: 3일~30일 (반복·중간 위반)
3. **영구 정지**: 중대 위반 또는 3회 이상 반복
4. **즉시 영구 정지**: 미성년자 관련 성적 콘텐츠, doxxing

---

## §6 이의제기 프로세스

- **이의제기 기한**: 조치 통보 후 14일 이내
- **방법**: gmdqn2tp@gmail.com에 "이의제기:" 제목으로 이메일
- **재검토 기간**: 7일 이내
- **재검토 결과**: 취소·유지·감경 중 하나 (최종 결정)

---

## §7 방장(호스트)의 책임

- 방 내 분위기 조성 및 규칙 위반자 퇴장 권한
- 방 내 규칙을 방 설명에 명시할 수 있음
- 방장도 이 가이드라인 위반 시 동일하게 제재

---

## §8 AI 생성 영상 특별 정책

- fal.ai API의 콘텐츠 정책 자동 적용 (프롬프트 사전 검사)
- 생성 후 다른 사용자가 신고한 영상도 검토 대상
- 영상 공개 범위(public/members_only/private)에 관계없이 신고 가능
