03 · Key Flows

핵심 흐름 3종번호 순서 = 실제 실행 순서 · 빨강 = 서버 게이트

A방 입장 → 라이브 연기

로그인→ 로비 public_rooms→ 입장 정원·연령 게이트→ livekit-token 참가자 검증→ LiveKit 연결→ 웹캠·음성·채팅·아바타 동기

B아바타 렌더 (Vtube 공장 → ChatterBox 무대)

AUTORIG 공장 PNG→rig→ character.json + PNG 파츠→ Supabase Storage avatars/→ PixiJS v8 네이티브→ 표정 드라이버 52ch→Cubism→ 라이브 얼굴

C더빙 파이프라인 5단계

① MP4 업로드→ ② STT Whisper→ ③ 역할 배정→ ④ 파트별 녹음 synced→ ⑤ 합성 fal 원음제거 + ffmpeg.wasm→ 완성본 다운로드
