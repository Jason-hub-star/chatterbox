# 도그푸딩 라운드 1 — 지인 초대 실사용 (2026-07-08 준비 완료)

로비 수직기능 페이즈루프(GAP-MATRIX Phase 1~7) 직후의 첫 실사용 라운드. 목적 = **기능 검증이 아니라 마찰 수집**(기능은 프리플라이트로 이미 실증). 여기서 나온 마찰이 다음 백로그의 원천.

## 프리플라이트 (기계 검증 완료 — 2026-07-08)

- **지인 전체 여정 9/9 PASS**(프로드, 헤드리스): 미가입자가 초대 URL 클릭 → /login 리다이렉트 → 회원가입 발견 → **UI 가입 즉시 로그인**(`mailer_autoconfirm: true` 확인 — 메일 확인 불요) → 초대 배너 → 분장실 → 무대 연결 → 채팅 왕복 → 퇴장.
- 로비에 테스트 방 잔재 없음(@e.com 13개 ended 처리됨). pg_cron 2종(예약 리마인더·vgen) 프로드 등재 확인.
- 알려진 표시 특성: 분장실에서 아바타(MediaPipe) 로딩 중 [무대로 입장]을 누르면 로딩이 끝날 때까지 화면 전환이 잠깐 지연될 수 있음(실기기 ~1초 — 관찰 대상 ①).

## 세션 플랜 (권장 40~60분, 지인 1~2명)

1. **초대**: 내 방 생성(장르 선택) → 관리 탭 [초대링크 만들기] → 아래 메시지와 함께 전송.
2. **입장 관찰**: 지인이 가입→분장실→무대까지 오는 동안 **어디서 머뭇거리는지** 시간 재기(목표: 링크 클릭→무대 3분 내).
3. **공연 코어 루프**: 음성 대화 + 표정 아바타 + 리액션(우클릭/길게 누르기) + 대본 진행 + 채팅.
4. **관전**: 셋째 사람(또는 모바일 폰)으로 [관전] 입장 — 폰에서 무대가 보이는지.
5. **소셜 루프**: 지인 퇴장 → 내가 관리 탭 "최근 함께한 사람" [초대 보내기] → 지인 로비 벨 알림 → 재입장.
6. **예약**: 로비에서 다음 세션 예약(+대상 체크) → 지인 벨에 예약 알림 확인.
7. (여유 시) 연습 무대 입장 → 대본 혼자 진행.
