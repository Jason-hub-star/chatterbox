Design system for "ChatterBox", a VTuber performing/collaboration platform.
Mood: dark, focused, community game-hub energy (Discord-inspired neutral base) — warm amber
used sparingly as the sole brand accent, never as a background wash.

Color tokens:
- background base: #0B0B0D (near-black, neutral gray)
- background panel/card: #18181C (dark gray)
- background elevated (modal/hover): #222227
- border: #2E2E35
- text: #F5F5F2 (off-white)
- text muted: #9C9CA3
- accent (brand/host/CTA — sparing use only): #FF8C2A (fire-amber)
- accent hot (recording/live/danger): #FF4500 (fire-hot)
- success/tracking: #56F09F (spring-green)

Typography:
- display: Schibsted Grotesk, weight 400-500
- body: Noto Sans JP

UI rules:
- Neutral dark gray/black base throughout — no colored ambient glow or tinted backgrounds
- Amber accent appears only on small elements (buttons, active/host indicators, icons)
- Large rounded panels (24-40px border radius)
- Pill-shaped primary buttons
- Avoid purple-blue gradients and generic AI-default aesthetics
- Avoid Heroicons defaults; use custom icons
- Japanese-first copy, support Korean/English
- Generous whitespace, high-contrast text

Desktop Stage Manager Overlay — host-only control surface for ChatterBox RoomView,
the primary surface during a live performance. 16:9, single state (mid-performance, normal mode).
Three-column layout:
Left column (~22%): Participants list — 6 rows, each showing avatar thumbnail, slot number,
role badge, mic/camera icon state, tracking signal bar, connection quality dot, drag handle.
Center column (~56%): Main stage preview — mode badge "NORMAL" (pill, fire-amber), current cue
text + next cue preview below, red "● REC 00:12:34" recording badge top-right.
Right column (~22%): tabbed feed — chat messages with floating reaction emojis, VGen job status
row "Generating... 42%", safety alert banner (collapsed, fire-hot border).
Bottom bar (full width, ~9%): soundboard 2x4 icon grid, record toggle (pill, fire-hot when active),
cue prev/next arrows with counter "3/8", VGen trigger button, rehearsal/performance mode switch.
Neutral #0B0B0D base, fire-amber accents on active/recording states only, off-white text,
rounded panels, no purple gradients.
