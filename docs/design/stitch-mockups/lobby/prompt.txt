Design system for "ChatterBox", a VTuber performing/collaboration platform.
Mood: dark, focused, community game-hub energy (Discord-inspired neutral base) — warm amber
used sparingly as the sole brand accent, never as a background wash.

Color tokens:
- background base: #0B0B0D (near-black, neutral gray)
- background panel/card: #18181C (dark gray)
- background elevated (modal/hover): #222227
- border: #2E2E35
- text: #F5F5F2 (off-white)
- text muted: #9C9CA3
- accent (brand/host/CTA — sparing use only): #FF8C2A (fire-amber)
- accent hot (recording/live/danger): #FF4500 (fire-hot)
- success/tracking: #56F09F (spring-green)

Typography:
- display: Schibsted Grotesk, weight 400-500
- body: Noto Sans JP

UI rules:
- Neutral dark gray/black base throughout — no colored ambient glow or tinted backgrounds
- Amber accent appears only on small elements (buttons, active/host indicators, icons)
- Large rounded panels (24-40px border radius)
- Pill-shaped primary buttons
- Avoid purple-blue gradients and generic AI-default aesthetics
- Avoid Heroicons defaults; use custom icons
- Japanese-first copy, support Korean/English
- Generous whitespace, high-contrast text

Desktop lobby page for finding and creating rooms.
Layout: top navigation bar with logo, user avatar, create-room CTA. Main area: filter bar (search input, genre multi-select badges, language dropdown). Below: grid of room cards. Each card shows room title, genre tag badges, current/max participants (e.g. "4/6"), private lock icon, language flag, live/waiting status dot, small overlapping avatar dots of participants, "Join" button. Bottom section: invite-code input field.
Cards use dark-gray #18181C background with rounded corners. Fire-amber accents for live status and primary buttons only.
