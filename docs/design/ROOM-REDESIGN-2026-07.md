## 0. BLUF

리디자인 = **레이아웃 셸 재편(2열 → 3존) + 기존기능 재배선 + 미빌드 컨셉요소는 시각 스텁/defer.**
컨셉의 ~10개 요소 중 실제 코드에 있는 건 절반(스크립트·리액션·마이크·나가기·메인뷰·탭·네트워크·씬배경·ProgressBar). 나머지 절반(분위기 지수·사운드보드 SFX오디오·룸 녹음·다음순서 큐·카메라/헤드폰 토글·채팅 오버레이 버블·영상 스크러버)은 **백엔드 신규**라 이번 MVP는 "컨셉처럼 보이되 기존기능이 새 셸에서 다 동작" + 미빌드는 시각 스텁으로 자리만 잡는다. 4 페이즈: **R1 셸+상/하단바 → R2 원형무대 → R3 좌측도크 → R4 우측도크.**

- **그리드:** `grid-rows: auto 1fr auto` / main `grid-cols: clamp(240px,20vw,280px) 1fr 300px`, `gap-3`.

- **토큰:** 배경 `stage-base`, 도크/바 `stage-panel`, 카드/모달 `stage-elevated`, 경계 `stage-border`, 본문 `stage-text`/보조 `stage-text-muted`. 액센트 `fire-amber`(호스트·활성·CTA만), `fire-hot`(라이브·녹음), `spring-green`(트래킹·ON). **앰버 배경워시 금지.**

## 3. 페이즈 플랜 (phase-loop, 각 페이즈 게이트 통과)

- **R1 — 셸 + 상/하단바:** `RoomPage`를 3존 그리드로 재편. 상단바(제목·태그·라이브타이머·초대·링크·N/6·⋮·네트워크), 하단 컨트롤바(마이크·스크립트·리액션·나가기 실배선 / 헤드폰·녹음·아바타 시각). **골격.**
- **R2 — 원형 무대:** 3×3 → 원형 3쌍 배치, 원형 아바타, 호스트 앰버링+크라운, 발화자 글로우+웨이브+뱃지, 마이크/트래킹 도트, 씬배경 히어로화, 별자리 글로우(lite).
- **R3 — 좌측 도크:** `ScriptPanel` 이식 + 세션정보(파생). [다음순서 큐 = defer]
- **R4 — 우측 도크:** 라이브노트(채팅) + 사운드보드 그리드(기존 리액션 재사용) + 분위기 지수(시각 스텁). 기존 DUB/VGen/관리 탭 보존.

> **as-built (2026-07-10, Fable — 인수 phase-loop):** R1 셸 커밋(OPS-1). **R2** 무대(호스트 앰버링·크라운·별자리 글로우·발화 글로우·이름칩·감독태그) `7ebf613`·`bd97384`. **R3** 좌도크 세션정보 카드(장르·러닝타임·언어 실데이터·Intl.DisplayNames) `bd97384`(다음순서 큐 defer). **R4** 우도크 카드 스택 `e442e61`: 방분위기=리액션 활기 **실집계 라이트**(활성 리액션 수+발화, 가짜 아님)+발화 파형(장식) / 사운드보드=slots 2×3→sendReaction 재사용 / RightPanel **도구탭 전 보존**(chat·dub·vgen·notes·host — 주인님 콜=카드스택+도구탭). 디렉터노트 배선(ROOM-17, chat 채널 message_type=note·author=participant 위조차단) `7ebf613`. **defer(후속):** 초대하기 상단 승격·라이브 타이머 완전 재배치·하단 스텁 배선(헤드폰/녹음/아바타)·다음순서 큐·채팅 오버레이 버블·영상 스크러버·mood 백엔드 집계·사운드보드 SFX. 게이트 tsc0·lint0·test120/120·docs:check PASS. 배포=프론트 CF Pages만(엔진 무변경).
