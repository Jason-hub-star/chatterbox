01. 한 줄 요약
핵심 메시지 그림 한 장으로 누구나 살아 움직이는 버튜버가 되고, 친구들과 실시간으로 함께 연기·더빙·쇼츠를 만드는 협업 버추얼 스튜디오
02. 문제 정의 — 왜 지금 이 문제인가
①

진입 장벽이 너무 높다

VTuber 캐릭터 리깅·라이브 셋업에 전문가·고가 장비가 필요해 소규모 창작자·성우·교육자가 도전하기 어렵다.

②

협업 도구가 분산되어 있다

여럿이 연기·더빙하려면 통화 툴·OBS·모델링 소프트웨어를 별도로 병행해야 해 창작 흐름이 끊긴다.

③

표현 욕구는 있지만 방법이 없다

'나만의 캐릭터로 말하고 싶다'는 욕구는 있지만 실현 방법이 없어 일반 화상통화로 대체하고 있다.

03. 솔루션 — ChatterBox가 해결하는 방식

🎭 PNG → VTuber 자동 생성

그림 1장 업로드 → AI 분해·리깅 → 웹캠 60fps 실시간 구동. 라이브 중 AI 추론 비용 0, OBS 연동 즉시 방송.

🎙️ 영상 더빙 공동 제작

영상 업로드 → AI 대본 자동 추출 → 참가자들이 역할 맡아 함께 더빙 녹음. 내 캐릭터로 말하는 경험.

🎬 쇼츠 공동 제작

다 같이 프롬프트 작성 → AI 영상 생성 → 즉시 녹음. 유튜브·릴스·틱톡 규격 바로 출력.

✨ 실시간 리액션

특정 장면에 이모지·체크·물음표 반응을 캐릭터로 실시간 소통. 전체 방에 즉시 동기화.

😊 MediaPipe 52채널 표정 동기화

설치 없는 브라우저 기반. 웹캠만 있으면 내 표정이 캐릭터에 그대로 반영.

🖥️ 설치 없는 브라우저 기반

별도 소프트웨어 불필요. URL 하나로 입장, 링크 공유로 최대 6인 협업 즉시 시작.
