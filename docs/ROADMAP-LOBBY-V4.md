# 로비 v4 로드맵 — 관별 기능 고도화 + UIUX 고도화

> **BLUF:** 로비 v3 재편(공연/제작/사람/나 4축)으로 **골격**은 섰다. 다음은 각 관의 **기능 두께**를 채우고(트랙 1) 횡단 **UIUX 폴리시**(트랙 2)를 얹는 것. 우선순위 1번은 **방금 개통한 쇼츠 제작소 완성**(VGEN-11/12) — 골격만 있고 실사용 가치가 UI 한 겹에 걸려 있어 ROI 최상.

## 완료 — 로비 v3 IA 재편 (2026-07-09, 배포됨)

- **공방 → 쇼츠 제작소**: VGEN 이 room 강결합이라 유저당 숨겨진 스튜디오 방(`is_studio`, `ensure-studio-room` 멱등, `public_rooms` 제외) 재사용으로 로비 독립 쇼츠 생성 — 스키마·보안(SEC-2/3) 무변경. `VgenStatusTab(roomId)` 재사용.
- **대극장 = 무대 통합**: 방 만들기(`StageCreator`)를 공방에서 흡수 — 목록·입장·예약·만들기가 한 건물.
- **찻집 = 사람 전용**: 방 재입장(대극장 중복) 제거, 사람 칩(재초대)만. SNS 커뮤니티 자리 선점.
- **모바일 배경 광장 통일**: 레거시 거리(`lobby-street/day`) → `plaza-1` (데스크톱과 동일). 커밋 `91007e8`·`74b0b70`, 프로드 실측(스튜디오 E2E 5/5·모바일 360 실렌더).
