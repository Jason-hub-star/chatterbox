# 상태 — 스냅샷 + 다음 액션 추천 (읽기 전용 · 공통 메타)

역할: "어때?"에 한 번에 답한다. **읽기만** 한다 — 코드·문서·배턴을 바꾸지 않는다(실행은 `/다음`).

## 훑을 것 (재읽기 낭비 금지)
- SessionStart 훅이 넣은 SCOUT 요약이 컨텍스트에 있으면 재읽기 금지. 없을 때만 `docs/status/SCOUT.md` 첫 `state:`.
- `docs/DOGFOOD-AUDIT-2026-07.md` §0 **트랙 A** 첫 `[ ]` 미체크 항목.
- `docs/GAP-MATRIX.md` P0 상태.
- 작업 상태: `git status -s`(미커밋)·`git log -1 --oneline`·미배포분 유무.

## 출력 (BLUF 1개)
- 첫 줄: 지금 상태 한 줄(배턴 state · 미커밋 유무 · 마지막 한 일).
- 둘째: **다음 액션 1개** 추천 — 항목 id·probe 경로·"왜 이게 다음인지". 여러 개 나열 금지.
- 실행이 필요하면 "`/다음` 하면 착수" 한 줄로 끝(여기서 실행하지 않음).

## 경계
- 공통 메타 명령. 어느 프로젝트든 `/상태`=읽기 스냅샷+추천으로 역할 고정(전역 CLAUDE.md 커맨드 규약).
