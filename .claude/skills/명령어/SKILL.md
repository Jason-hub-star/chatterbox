# 명령어 — ChatterBox 명령 표

아래 표를 장식 없이 그대로 출력한다. **공통 5**는 모든 프로젝트에서 이름·역할 동일, **도메인**은 ChatterBox 전용.

## 공통 (어느 프로젝트든 동일)
| 명령 | 하는 일 | 읽기/쓰기 |
|---|---|---|
| `/상태` | 지금 어때 + 다음 액션 1개 추천 | 읽기 |
| `/다음` | 추천 말고 바로 다음 백로그 항목 실행(구 /backlog·가장 에이전틱) | 쓰기 |
| `/진단 <증상>` | 고장·막힘 런북(근본원인 1개+확인명령) | 읽기→고침 |
| `/마감` | 게이트·문서·SCOUT 배턴·커밋 마무리 | 쓰기 |
| `/명령어` | 이 표 | 읽기 |

## ChatterBox 도메인
| 명령 | 인자 | 하는 일 |
|---|---|---|
| `/배포` | `[이름…]` | 프로드 배포→2탭 E2E→백로그[x]·GAP·drift(구 bepo) |
| `/감사` | — | 도그푸딩 다중 페르소나 보안·UX 감사(구 dogfood-audit) |
| `/루프` | `[§0 섹션·브리프 경로·발견]` | 발견 묶음→골 사다리 조립→승인→phase-loop 완주(플랜 대조·docs:drift 앵커 자동) |

세부 절차는 각 명령의 스킬 문서. 자연어로 말해도("배포해줘", "상태 어때") 트리거가 잡는다. 규약 SSOT=전역 `~/.claude/CLAUDE.md` 커맨드 규약.
